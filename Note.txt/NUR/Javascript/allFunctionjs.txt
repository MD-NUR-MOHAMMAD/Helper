JavaScript has many **built-in functions**, and you can create your own **custom functions**. If you're asking for **all JavaScript functions**, it's important to clarify that JavaScript has:

1. **Global functions**
2. **Object methods**
3. **Standard library functions (Math, String, Array, etc.)**
4. **User-defined functions**

Here’s a categorized list of the **most commonly used JavaScript built-in functions and methods**.
